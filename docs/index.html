Fan Club MkIV

基于STM32的智能风扇控制系统

快速开始 GitHub

项目概述

🎯 项目目标

开发一个高性能、可扩展的智能风扇控制系统，支持多种控制模式和实时监控功能。

🔧 技术栈

  • STM32F429ZI微控制器
  • Mbed OS实时操作系统
  • C++面向对象编程
  • PWM信号控制

📊 应用场景

适用于工业自动化、智能家居、数据中心散热等需要精确风扇控制的场景。

功能特性

⚡

高精度PWM控制

支持0-100%无级调速，响应时间小于10ms

📡

串口通信

115200波特率串口通信，支持实时命令控制

🔄

实时监控

实时监控风扇状态、转速和系统参数

🛡️

安全保护

过载保护、温度监控、故障自动恢复

🔧

模块化设计

组件化架构，易于扩展和维护

📈

性能优化

低功耗设计，高效率控制算法

系统架构

核心组件

  • Fan类 - 风扇控制核心逻辑
  • Communicator类 - 串口通信处理
  • Processor类 - 数据处理和算法
  • FastPWM库 - 高精度PWM信号生成

快速开始

1

环境准备

# 安装Mbed CLI
pip install mbed-cli

# 安装ARM GCC工具链
# 下载并安装 GNU Arm Embedded Toolchain
2

克隆项目

git clone https://github.com/your-username/fan-club-master.git
cd fan-club-master/slave
3

编译项目

# 编译固件
mbed-tools compile -t GCC_ARM -m NUCLEO_F429ZI

# 或使用提供的脚本
.\compile_slave.bat
4

烧录固件

# 将生成的.bin文件拖拽到开发板
# 或使用ST-Link Utility烧录

API文档

Fan类 API

setSpeed(int speed)

设置风扇转速

  • 参数: speed - 转速百分比 (0-100)
  • 返回: bool - 设置是否成功
Fan fan;
fan.setSpeed(75); // 设置转速为75%

getSpeed()

获取当前风扇转速

  • 返回: int - 当前转速百分比
int currentSpeed = fan.getSpeed();

Communicator类 API

sendMessage(string message)

发送串口消息

  • 参数: message - 要发送的消息
  • 返回: bool - 发送是否成功
